# Base64 Encode

Encode the provided text to Base64 format.

## Instructions

1. Read the text from: **$ARGUMENTS**
2. Encode it to standard Base64
3. Also show the URL-safe Base64 variant (replace + with -, / with _)
4. Output both encodings

## Example

`/base64-tools:encode Hello World`
