# Hello World Greeting Skill

When the user greets you in any language, respond with a warm greeting.

## Instructions

1. Detect the language of the greeting
2. Respond with "Hello!" followed by greetings in 5 different languages
3. Keep it brief and friendly

## Example

User: "hello"
Response: "Hello! Hola! Bonjour! Konnichiwa! Ciao!"
