# Lorem Ipsum Generator

Generate placeholder text for testing and prototyping.

## Instructions

1. Parse the number of paragraphs from: **$ARGUMENTS** (default to 1 if not specified)
2. Generate that many paragraphs of lorem ipsum text
3. Start with the classic "Lorem ipsum dolor sit amet..."
4. Output as plain text with blank lines between paragraphs

## Example

`/lorem-ipsum:generate 3`
