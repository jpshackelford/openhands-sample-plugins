# React + Vite Project Scaffold

Create a new React project using Vite.

## Instructions

1. Read the project name from: **$ARGUMENTS**
2. Run `npx create-vite@latest <project-name> --template react-ts`
3. Report the created structure
4. Suggest next steps (cd into directory, npm install, npm run dev)

## Example

`/project-init:react my-app`
